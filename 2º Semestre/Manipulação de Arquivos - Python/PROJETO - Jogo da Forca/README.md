# Jogo da Forca em Python

Utilizando a linguagem de programa Python e manipulação de arquivos, foi feito um jogo da forca, onde a palavra é sorteada de um arquivo chamado "forcapalavras.txt", e o usuário tenta acertar. Ao perder mostra o boneco enforcado.
